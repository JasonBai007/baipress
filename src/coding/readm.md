## How to use

``` bash
# install dependencies
yarn install

# serve with hot reload at localhost:8080
yarn run dev

# build for production with minification
yarn run build

# preview your dist files
cd dist
http-server

# stop your local http server
Ctrl + C

# analysis your dist files
yarn run analysis

```
